## 3. СТОП-СЛОВА -- чего избегать в коммуникации

### Маркетинговые штампы EdTech
| Фраза | Почему нельзя |
|-------|---------------|
| "Профессия будущего" | Затёрто до потери смысла. Так пишут все с 2018 года. |
| "Войти в IT" | Мем. Вызывает ироничную реакцию у ЦА, которая уже начитана. |
| "Обучение с нуля" | Подразумевает, что человек -- ноль. Снижает самооценку. |
| "Диплом государственного образца" | У Quillon его нет. И он не нужен -- работодатели смотрят портфолио. |
| "Гарантия трудоустройства" | Невыполнимое обещание, ассоциируется с обманом. ЦА уже не верит. |
| "Освойте новую профессию" | Пустая формула без конкретики. Кто угодно может написать то же самое. |
| "Топовые эксперты / практикующие специалисты" | Каждая школа это пишет. Ноль дифференциации. |
| "Обучение в удобном темпе" | = никто не контролирует = я брошу через месяц. Для ЦА это красный флаг. |
| "Станьте востребованным специалистом" | Абстрактное обещание. Не отвечает на вопрос "как именно?". |
| "Погрузитесь в атмосферу" | Маркетинговая вода. Человек хочет навык и оффер, а не атмосферу. |

### Затёртые конкурентами фразы
| Фраза | Кто использует |
|-------|---------------|
| "Учим на практике" | Яндекс.Практикум (их слоган). Повторять = быть копией. |
| "Первая работа через N месяцев" | Skillbox, GeekBrains. Воспринимается как ложь. |
| "Персональный ментор" | Все. При этом у большинства это чат-бот или перегруженный ревьюер. |
| "Программа от лидеров индустрии" | Нетология, Skillbox. Расплывчато, не проверяемо. |
| "Скидка 40% только сегодня" | Все. ЦА знает, что "скидка" будет и завтра. Убивает доверие. |

## 4. СЛОВАРЬ ДОВЕРИЯ -- что использовать

### Слова и фразы доверия для ЦА

| Слово / фраза | Почему работает |
|---------------|-----------------|
| Реальный продукт | Противопоставляет учебным проектам. Конкретно, проверяемо. |
| Коммерческий проект | Язык работодателя. Звучит как строчка в резюме. |
| Кросс-команда | Показывает, что работа = не в одиночку. Совпадает с реальным IT. |
| Полный возврат за 7 дней | Снимает финансовый страх. Прямо, без звёздочек. |
| С 21-й недели | Конкретный срок = доказательство плана. Нельзя подделать. |
| 8 250 руб/мес | Цена видна сразу. Честность = доверие. |
| ChatAI, Meet, LMS | Названия продуктов = конкретика. Можно проверить, найти, потрогать. |
| 12 месяцев | Конечный срок. Не "в удобном темпе", а "вот дедлайн". |
| 10 000+ пользователей | Масштаб продукта. Не учебная песочница. |
| Без увольнения с работы | Прямо отвечает на страх "а как совмещать". |

### Глаголы действия

| Глагол | Контекст использования |
|--------|----------------------|
| Работать | "Работать на реальном продукте", не "изучать теорию" |
| Писать код | Конкретное действие, не абстрактное "осваивать навыки" |
| Коммитить | IT-язык. Показывает: здесь всё по-настоящему. |
| Деплоить | Ваш код попадает в продакшн, а не в папку на рабочем столе. |
| Проходить ревью | Процесс реальной разработки. Сразу понятно: здесь как на работе. |
| Решать задачи | Не "выполнять домашку", а решать задачи бизнеса. |
| Зарабатывать | Конечная цель. Можно использовать с конкретной цифрой. |
| Показать на собесе | Привязка к результату -- трудоустройство. |
| Совмещать | "Совмещать с работой" -- снимает возражение про время. |
| Выбрать трек | Конкретный первый шаг, снижает тревогу выбора. |

### Метрики, которые важно использовать

| Метрика | Значение | Зачем |
|---------|----------|-------|
| Срок до реальных проектов | 21 неделя (6-й месяц) | Отсечка: "здесь вы не будете год учить теорию" |
| Количество продуктов компании | 3 (ChatAI, Meet, LMS) | Конкретные названия = проверяемость |
| Стоимость в рассрочку | 8 250 руб/мес | Сравнение с привычными тратами (подписки, кофе, такси) |
| Пробный период | 7 дней, полный возврат | Убирает страх "а если не моё" |
| Нагрузка в неделю | 12 часов | Показывает: совмещать реально |
| Количество треков | 3 (Python, Flutter, QA) | Выбор ограничен = проще решиться |
| Полная стоимость | 198 000 руб | Открытая цена = честность |
| Старт коммерческих проектов | С 6-го месяца | Строчка в резюме ещё до окончания программы |
