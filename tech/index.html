Стек

Стек, на котором мы строим

Каждая технология выбрана по одному критерию: решает ли она задачу лучше альтернатив. Не потому что модно. Потому что работает.

Python 3.11+ FastAPI Django 5 DRF PostgreSQL Redis Docker

Python — единый язык для backend, AI/ML и автоматизации. FastAPI для высоконагруженных асинхронных API с автодокументацией. Django для сложных CRUD-систем: LMS, CRM, административные панели. PostgreSQL — ACID-транзакции, JSON-поля, масштабируемость без компромиссов.

Quillon Meet API обрабатывает сигнализацию WebRTC и управление комнатами. LMS API обслуживает 4 роли пользователей с разными уровнями доступа. Всё на Python.

Flutter Dart React Native Firebase REST API

Flutter — один код на iOS и Android с нативной производительностью. Рынок Flutter-разработчиков в России в 4 раза меньше, чем React-разработчиков, при сопоставимом количестве вакансий. React Native используем в Meet для интеграции с LiveKit SDK.

LMS доступен как PWA на мобильных устройствах. Meet — полноценное нативное приложение для iOS и Android с камерой, микрофоном и push-уведомлениями.

RAG-системы Fine-Tuning LLM ИИ-агенты Prompt Engineering Vector DB

AI — инфраструктурный компонент, не маркетинговая надстройка. RAG-архитектура для работы с корпоративными данными без переобучения моделей. ИИ-агенты для автоматизации инженерных и бизнес-процессов. Собственные пайплайны промптов с версионированием и A/B-тестированием.

Quilly AI — корпоративный ассистент с RAG-пайплайном и векторным поиском. AI-персонализация в воронке продаж: автоматическая генерация видео-разборов для каждого лида.

Pytest Selenium Playwright Allure Appium

Автотесты — не опция, а часть CI/CD. Pytest для unit и integration тестов бэкенда. Playwright для E2E-тестов веб-интерфейсов: быстрее Selenium, кросс-браузерный. Appium для мобильных приложений. Allure — единый дашборд отчётов для всей команды.

Каждый PR в репозиторий проходит автоматическую проверку. Тесты запускаются до merge, а не после. Инженеры QA-трека пишут реальные тест-кейсы для Meet и LMS с первого дня.

Nginx Let's Encrypt SSL Docker Compose GitHub Actions S3

Своя инфраструктура — контроль над данными, предсказуемые расходы, независимость от облачных провайдеров. Nginx как reverse-proxy для всех сервисов. Docker для изоляции и воспроизводимости окружений. GitHub Actions для CI/CD без vendor lock-in.

Один сервер (Selectel, Россия) хостит Meet, LMS, Git, Cloud, Mail. Деплой Meet: git pull, docker compose build, docker compose up — 3 минуты от коммита до production.

Принципы

Как мы строим

Пять принципов, которые определяют инженерную культуру Quillon.

01
Production-first

Мы не делаем учебных проектов. Каждый продукт работает в production с реальными пользователями. Код, который пишет команда, деплоится на серверы и обслуживает людей. Это формирует другой уровень ответственности и качества.

meet.quillon.ru — его можно открыть прямо сейчас, создать комнату и провести звонок.

02
AI-native

ИИ — не дополнение к процессу, а его основа. Код-ревью с AI, автогенерация тестов, документация через LLM, RAG для поиска по базе знаний. Мы не учим промпты — мы строим продукты, где AI является инфраструктурным компонентом.

Quilly AI использует RAG-пайплайн для поиска ответов в документах компании.

03
Own your stack

Минимум зависимости от третьих сторон. Свои серверы, свой Meet, свой LMS, свой Git-хостинг, своя почта. Данные и инфраструктура полностью под нашим контролем.

Когда Zoom и Google Meet стали недоступны в России, мы не искали замену — у нас уже был свой Meet.

04
Ship fast, fix faster

Короткие циклы разработки. Деплой несколько раз в день. Ошибки неизбежны — важна скорость их обнаружения и устранения. CI/CD — не модное слово, а ежедневная реальность.

Деплой Quillon Meet: git pull, docker compose build, docker compose up. Весь цикл — 3 минуты.

05
Code review culture

Каждый коммит проходит ревью. Не формальный approve, а разбор по существу: архитектура, производительность, читаемость, безопасность. Это создаёт единый стандарт качества кода по всей компании.

PR в репозиторий LMS проходит автоматические тесты, затем ревью от senior-инженера.
