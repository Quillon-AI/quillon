ПРИНЦИПЫ

Как мы строим

Пять принципов, которые определяют инженерную культуру Quillon.

Production-first

Каждый продукт проходит полный цикл: архитектура, код, CI/CD, мониторинг, поддержка. Мы отвечаем за решение от первого коммита до стабильной работы под нагрузкой.

meet.quillon.ru — открой и проведи звонок прямо сейчас.

AI-native

ИИ — не дополнение к процессу, а его основа. Код-ревью с AI, автогенерация тестов, документация через LLM, RAG для поиска по базе знаний.

Quilly AI на RAG-пайплайне для поиска по документам компании.

Own your stack

Минимум зависимости от третьих сторон. Свои серверы, свой Meet, свой LMS, свой Git-хостинг, своя почта. Ни один внешний сервис не может парализовать работу компании.

Когда Zoom стал недоступен — переход занял 0 дней.

Ship fast, fix faster

Короткие циклы разработки. Деплой несколько раз в день. Ошибки неизбежны — важна скорость их обнаружения и устранения. CI/CD на каждом продукте.

Code review culture

Каждый коммит проходит ревью. Не формальный approve, а разбор по существу: архитектура, производительность, читаемость, безопасность.

Hybrid infra

Dedicated-серверы в РФ для production + облако для CDN и масштабирования. 152-ФЗ, предсказуемая стоимость, независимость от провайдера.

FAQ

Частые вопросы

Сколько стоит разработка проекта?

Зависит от стека и сложности. Telegram-бот с интеграцией платежей — от 200 000 ₽, MVP мобильного приложения на Flutter — от 800 000 ₽, AI-интеграция с RAG — от 500 000 ₽. Бесплатная техническая оценка после первого созвона: разбираем ТЗ, прикидываем стек, даём диапазон бюджета и сроки.

Какой стек используете?

Backend: Python 3.11+, FastAPI, Django 5, PostgreSQL, Redis, Celery. Mobile: Flutter, Dart. AI: RAG, LLM API (OpenAI, Anthropic, локальные), Vector DB (Pinecone, Qdrant). Infrastructure: Docker, Nginx, GitHub Actions, S3, dedicated-серверы в РФ. Frontend: Vue 3, TypeScript, Vite. Полный стек — на главной странице.

Как начать сотрудничество?

Заполните форму ниже или напишите в Telegram @quillon. Опишите задачу — что нужно сделать, желательные сроки, бюджетный коридор, ссылка на ТЗ если есть. Ответим за день, договоримся о 30-минутном созвоне для оценки.

Можно ли увидеть код в production?

Да. Часть кода — в open-source: github.com/Quillon-AI. Доступны репозитории Quillon Meet (web, desktop, mobile), отдельные backend-инструменты. По запросу можем показать архитектурные решения коммерческих проектов под NDA.

Берёте проекты под ключ или работаете по часам?

Оба варианта. Под ключ — фиксированный scope, фиксированная цена, риски на нашей стороне. По часам — time&material для продуктов с долгосрочной поддержкой или scope, который сложно зафиксировать заранее. По факту чаще всего гибрид — фикс на первый этап, T&M на поддержку.

Подписываете NDA?

Да. Стандартный шаблон NDA отправляем в первый день общения, можем подписать перед получением материалов. Если у вас свой шаблон — подстраиваемся.
