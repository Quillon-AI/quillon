01 / Видеоконференции

Quillon Meet

Платформа видеоконференций с адаптивным стримингом, записью и кросс-платформенными клиентами.

Задача

Zoom и Google Meet недоступны в России. Командам нужен надёжный инструмент для ежедневных созвонов и вебинаров без зависимости от зарубежных сервисов.

Решение

Собрали платформу на WebRTC с собственным сигнальным сервером. Простая ссылка-приглашение, без регистрации, P2P для малых групп, SFU для больших.

Платформы Web · Desktop · Mobile
Capacity До 50 участников · simulcast
Latency P2P + auto-reconnect
WebRTC Next.js Electron React Native Nginx Docker S3 FFmpeg
meet.quillon.ru ↗
02 / LMS

Quillon LMS

Платформа управления обучением с видео-лекциями, заданиями, аналитикой прогресса и ролевой системой.

Задача

Готовые LMS не дают контроля над UX, данными и интеграциями. Нужна платформа, которую можно развивать под свои процессы обучения.

Решение

Django + Vue 3 SPA. 4 роли: менеджер, преподаватель, куратор, участник. Тематизация через Origin-систему, PWA для mobile-доступа, прогресс-трекинг.

Роли Менеджер · Преподаватель · Куратор · Участник
Features Видео-лекции · задания · прогресс · PWA
Архитектура SPA + REST · Origin-тематизация
Django 5 DRF Vue 3 TypeScript Vite PostgreSQL Nginx
lms.quillon.ru ↗
03 / AI · корпоративный

Quilly AI

Корпоративный AI-ассистент с контекстным поиском по внутренней базе знаний и персонализацией под роль пользователя.

Задача

Обычные чат-боты галлюцинируют и не знают контекст компании. Нужен ассистент, который отвечает на основе реальных документов и цитирует источник.

Решение

RAG-пайплайн: индексация документов в vector DB, контекстный поиск, ответ через LLM с цитатами. Встроен в LMS, доступен через API.

Скорость Ответ за 2–4 секунды
Точность Цитирование источника
Интеграция LMS · API · Telegram
Python FastAPI RAG LLM API Vector DB Redis
04 / Автоматизация

Quillon Jobs

Система автоматического подбора IT-заказов с фриланс-бирж под уровень участника.

Задача

Джуну сложно найти первый заказ: нет портфолио, непонятно что брать, страшно облажаться. Нужен фильтр, который подберёт задачу по грейду.

Решение

Парсер собирает 500+ задач в месяц с нескольких источников. Алгоритм матчинга по стеку и уровню. Уведомления о подходящих заказах.

Volume 500+ задач/мес
Средний чек 5 000 — 30 000 ₽
Matching Авто по стеку и грейду
Python FastAPI Парсинг PostgreSQL Telegram Bot
